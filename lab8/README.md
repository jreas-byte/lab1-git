# Лабораторна робота 8

## Варіант 4

Потрібно створити додаток для перевірки пам'яті.

Користувач вводить дані, додаток перевіряє, чи було введено число.
Якщо дані вірні, додаток перевіряє, чи значення більше 70.
Якщо дані неправильні, програма викликає помилку.

## Встановлення залежностей

pip install -r requirements.txt

## Запуск програми

python main.py

## Запуск із Sentry у Windows PowerShell

$env:SENTRY_DSN="ВАШ_DSN_ІЗ_SENTRY"
python main.py

## Запуск із Sentry у Git Bash

export SENTRY_DSN="ВАШ_DSN_ІЗ_SENTRY"
python main.py

## Приклади введення

Правильне значення:

80

Результат:

Значення більше 70, пам'ять використовується помилка

Неправильне значення:

abc

Результат:

ValueError: Потрібно ввести число
