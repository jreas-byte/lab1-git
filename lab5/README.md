# Лабораторна робота 5

## Варіант 4

Створити додаток, який підраховує витрати води у комунальній сфері.

## Опис програми

У програмі реалізовано функції:

- calculate_usage — обчислює обсяг використаної води;
- calculate_bill — обчислює суму до оплати;
- create_summary — формує підсумок розрахунку.

## Запуск програми

python water_bill.py

## Запуск doctest

python -m doctest water_bill.py -v

## Запуск unit-тестів

python -m unittest test_water_bill.py
