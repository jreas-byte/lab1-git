# Лабораторна робота 7

## Варіант 4

Створити додаток для розрахунку зарплати працівників невеликої компанії.

## Опис

Основна логіка зберігається у функції `calculate_payroll`.

На вхід функція отримує список працівників типу `list[dict]`, де вказано:
- ім'я працівника;
- кількість відпрацьованих годин;
- погодинну ставку.

Функція повертає список типу `list[dict]` із нарахованою зарплатою кожного працівника.

## Запуск програми

python app.py

## Запуск doctest

python -m doctest app.py -v

## Запуск unit-тестів

python -m unittest test_app.py

## Встановлення залежностей

pip install -r requirements.txt

## Перевірка форматування

flake8 app.py test_app.py

## Форматування коду

black app.py test_app.py

## Налаштування Git Hook pre-commit

Для перевірки форматування перед комітом потрібно створити файл:

.git/hooks/pre-commit

Вміст файлу:

#!/bin/sh
echo "Перевірка коду за допомогою flake8..."
flake8 lab7/app.py lab7/test_app.py
if [ $? -ne 0 ]; then
    echo "Помилка: flake8 знайшов проблеми у коді. Коміт скасовано."
    exit 1
fi
echo "Перевірка пройшла успішно."
exit 0

Після створення потрібно зробити файл виконуваним:

chmod +x .git/hooks/pre-commit
